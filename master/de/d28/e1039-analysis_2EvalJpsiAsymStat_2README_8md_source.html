1 # e1039-analysis/EvalJpsiAsymStat
2 
3 A program to evaluate the statistical precision of J/psi TSSA.
4 It has been made from `AnaSimDst` under `knakano0524/e1039-analysis`.
5 It was moved to `E1039-Collaboration/e1039-analysis` in Jan. 2022.
6 
7 This version was used to produce the result presented on 2021-07-30.

10 ## Event Generation
11 
12 A normal set of J/psi simulated events has been already generated by Kenichi.
13 It is listed in `/pnfs/e906/persistent/users/kenichi/sim_data/jpsi_acc_20200914.list`.
14 The DST list (`AnaSimDst/work/list_dst.txt`) points to it be default.
15 Only when necessary, your own set can be generated with `SimChainDev`.

18 ## Event Analysis by `AnaSimDst`
19 
20 The simulated DSTs can be processed by `AnaSimDst`, as explained in `AnaSimDst/README.md`.
21 The conditions and qualities of simulated DSTs should be checked by looking into the outputs of `Fun4SimTree.C`.
22 
23 This branch includes another ROOT macro, `asymmetry.C`.
24 It reads the tree (like `Fun4SimTree.C`) to compute J/psi TSSA;
25 ```
26 root -b -q asymmetry.C
27 ```

30 ## Plot of Expected Accuracy
31 
32 Another ROOT macro in this branch, `draw_stat_exp.cc`, can be used to draw the expected accuracy of J/psi TSSA.
33 You first write the simulated accuracy (which is obtained by `asymmetry.C`) into this macro manually and then execute it;
34 ```
35 root -b -q draw_stat_exp.cc
36 ```
