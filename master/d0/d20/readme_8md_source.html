7 ### run the Fun4All shielding macro into the local gpvm machine:
8 ```
9 cd <path-to-/seaquest-analysis/E1039Shielding>
10 root -l Fun4E1039Shielding.C
11 ```
12 Several root files are generated.
13 The 'trk_eval.root' contains the information from the evaluator [`TrkEval`]. You may open that file and check the contents.

25 ### analysis
26 This root file 'trk_eval.root' needs to be attached in the macro for analysis. The analysis macro `ana.C` is for acceptance vs hole size study and the macro `loop_ana.C` is for acceptance vs dimuon mass study. You may test it using the example data from the trk_eval.root.
27 ```
28 root -l
29 root [0] .L ana.C
30 root [1] drawRelAcc()
31 root [2] drawAccPhi()
32 ```
33 
34 ```
35 root -l
36 root [0] .L loop_ana.C
37 root [1] drawAccMass()
38 ```
