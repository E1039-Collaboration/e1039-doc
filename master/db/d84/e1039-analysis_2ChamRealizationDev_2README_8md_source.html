1 # e1039-analysis/ChamRealizationDev
2 
3 A compact program to develop the chamber realization, which adds inefficiencies and position resolutions to chamber and prop tube hits.

6 ## Preparation
7 
8 The version of e1039-core is selected in `setup.sh`.
9 The pre-built offline version is used by default.
10 If you need modify e1039-core, you build e1039-core and select it in `setup.sh`.

14

23 
24 `AnaTrack` is a SubsysReco module to analyze reconstructed tracks.
25 The track reconstruction and the analysis are done before and after the chamber realization in order to evalaute the effect of the chamber realization.

28 ## Event Generation
29 
30 ```
31 cd macro
32 root -b 'Fun4AllStep1.C(10000)'
33 ```
34 
35 This step generates a set of single-track events using `PHG4SimpleEventGenerator`.
36 The vertex and the momentum of tracks are fixed by default for simplicity.
37 You can change them as you need, by modifying `Fun4AllStep1.C`.
38 
39 This step also includes the reconstruction and the analysis of tracks.
40 Results of the analysis will appear under `AnaTrack_raw/`.
41 The chamber realization is carried out in the next step.
42 
43 
44 ## Chamber Realization
45 
46 ```
47 root -b Fun4AllStep2.C
48 ```
49 
50 This step reads the DST file (`DST.root`) created in the previous step.
51 It applies the chamber realization and then reconstruct and analyze tracks.
52 Results of the analysis will appear under `AnaTrack_real/`.
53 
54 You compare `AnaTrack_raw/` and `AnaTrack_real/` to evaluate the effect of the chamber realization.
