1 # A Package to Generate and Analyze Hit-Embedded MC Events
2 
3 The hit embedding is a procedure for embedding a set of hits into a simulated physics event (like J/psi).
4 It is to mimic a real event, which contains not only a physics process but also background hits.
5 It can be used in studying the effects of background hits, such fake track and efficiency loss.
6 
7 ## Terminology
8 
9 For events, data or files:
10 
11 | Term | Meaning |
12 | ---- | ------- |
13 | "Signal" events | (Simulated) physics events |
14 | "Embedding" events | Events which (or whose hits) will be embedded |
15 | "Embedded" events | Physics events where hits have been embedded |
16 
17 ## Prerequisite
18 
19 This program depends on the E1039 resource+share+core packages.
20 You are recommended to use "spinquestgpvm01.fnal.gov".
21 You are expected to be familiar with the procedure for generating and analyzing a set of simulated events, using `SimChainDev` and `AnaSimDst` in `e1039-analysis` for example.
22 
23 You can check out the repository as usual;
24 ```
25 cd /path/to/your_working_directory
26 git clone git@github.com:E1039-Collaboration/e1039-analysis.git
27 cd e1039-analysis/HitEmbedding
28 ```

30 ## Software structure
31 
32 ### Source codes
33 
34 All source codes specific to this packages are stored in `src/`.
35 You have to compile them at least once by the following commands;
36 ```
37 source setup.sh
38 cmake-this
39 make-this
40 ```
41 
42 `cmake-this` and `make-this` can be executed at any directory, since they don't depend on the current directory.
43 They have to be executed again after you modify the codes.
44 
45 ### Fun4All macros
46 
47 Multiple Fun4All macros are available in this packages in order to carry out the hit embedding step-by-step.
48 Details of each macro are explained in `README.md` of each directory.
49 Probably you need not generate embedding events, since the E906 NIM3 events are already available and used by default.
50 
51 1. [`macro_gen_signal/`](macro_gen_signal/): Macro to generate signal events.
52 1. [`macro_gen_emb/`](macro_gen_emb/): Macro to generate embedding events.
53 1. [`macro_gen_emb_e906/`](macro_gen_emb_e906/): Macro to generate embedding events from E906 NIM3 events.
54 1. [`macro_embed/`](macro_embed/): Macro to do the hit embedding.
55 1. [`macro_ana/`](macro_ana/): Macro to analysis the hit-embedded events.
