1 # e1039-analysis/CalibChamXT
2 
3 A program to calibrate the chamber X-T curve, using cosmic-ray events at present.
4 The file structure, the configuration and the usage are similar to the following programs;
5 
6 * https://cdcvs.fnal.gov/redmine/projects/calib-cham-rt
7 * https://github.com/E1039-Collaboration/e1039-analysis/tree/master/AnaRealDst

12 ## Download
13 
14 You are recommended to use `spinquestgpvm01`.
15 You first clone the GitHub repository as usual;
16 
17 ```
18 ssh spinquestgpvm01.fnal.gov
19 cd /path/to/your/working/directory
20 git clone git@github.com:E1039-Collaboration/e1039-analysis.git
21 cd e1039-analysis/CalibChamXT
22 ```
23 
24 The top directory of this program is `e1039-analysis/CalibChamXT/`.
25 All files/directories explained below are relative to the top directory, unless specified.

28 ## Preparation
29 
30 The runs to be analyzed are listed in `work/list_run.txt`.
31 The list contains only run 3492 at present.
32 You could modify the list when necessary.
33 
34 You copy the DST files of all runs from `data2` to `pnfs` as follows.
35 It usually takes a few minutes per run;
36 
37 ```
38 cd work
39 ./copy_dst.sh
40 ```
41 
42 The source code is located in `src/`.
43 The following commands set up the shell environment and compile the code;
44 
45 ```
46 source setup.sh
47 cmake-this
48 make-this
49 ```

52 ## Track Reconstruction
53 
54 The program can execute the track reconstruction, not yet the calibration of the X-T curve.
55 
56 The following commands carry out a test, using 10,000 events in the 1st DST file.
57 It should take 10-20 seconds.
58 
59 ```
60 cd work
61 ./run_reco.sh -d 1 -e 10000 GO
62 ```
63 
64 The main output file is `scratch/main/run_003492_spill_000000000_spin/out/eval.root`.
65 The ROOT file (`eval.root`) contains all tracklets.
66 You had better check the number of tree entries is non-zero.
67 
68 You then execute the following commands to process all events/runs.
69 The `-g` option is to run jobs on the grid.
70 
71 ```
72 source /e906/app/software/script/setup-jobsub-spinquest.sh
73 kinit
74 ./run_reco.sh -g GO
75 ```
76 
77 You use `jobsub_q_mine` (or `./check_job.sh`) to wait until all grid jobs finish.
78

80 ## Tracklet Analysis
81 
82 Then the following commands analyze the tracklets reconstructed by `run_reco.sh`.
83 Plots will appear in `calib/1/`.
84 
85 ```
86 cd work
87 ./make_rt.sh
88 ```
