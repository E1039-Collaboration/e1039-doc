1 # e1039-analysis/AnaSimDst
2 
3 A compact program to analyze the simulated DST file, which is created by the `SimChainDev` package.
4 Since it is yet under development and also depends on other software under development,
5 it might stop working occasionally. In such cases, please contact the author.
6 
7 ## Analysis Flow
8 
9 1. DST -> Micro DST (uDST)
10  - By macro `Fun4SimDst.C`.
11  - To reduce the data size.
12 1. uDST -> Tree
13  - By macro `Fun4SimMicroDst.C`
14  - To further reduce the data size & calculate your custom quantities.
15 1. Tree -> Plots
16  - By macro `Fun4SimTree.C`
17  - To draw your quantities with various conditions.
18 
19 When the data size is small enough or the analysis logic is simple enough in your analysis, you can put all the steps in one macro.
20 
21 ## Basic Usage
22 
23 You can try the following procedure to go through the whole analysis flow.
24 
25 ### Preparation of Simulated DST Files
26 
27 You first produce a set of simulated DST files by `SimChainDev`.
28 This document assumes that the top directory of your simDST files is `/your/dst/dir`.
29 It can be either relative or absolute.
30 
31 The following command lists up all simDSTs into `list_dst.txt`;
32 ```
33 cd e1039-analysis/AnaSimDst/work
34 ./make-dst-list.sh /your/dst/dir
35 ```
36 You might manually create or edit the list file as you need.
37 
38 ### Compilation of Analysis Code
39 
40 The analysis code is located in `e1039-analysis/AnaSimDst/src`.
41 The following commands set up the shell environment and compile the code;
42 ```
43 cd e1039-analysis/AnaSimDst
44 source setup.sh
45 cmake-this
46 make-this
47 ```
48 You are expected to later look into the script and the analysis code to make your own analysis logic.
49 
50 ### Execution of Analysis
51 
52 The 1st step is to create uDST from DSTs.
53 The following commands reads five simDSTs for test.
54 The uDST file is `uDST.root`.
55 ```
56 cd e1039-analysis/AnaSimDst
57 source setup.sh
58 cd work
59 root -b -q 'Fun4SimDst.C(5)'
60 ```
61 
62 The 2nd step is to create Tree from uDST, by the following command.
63 The TTree object is outputted to `sim_tree.root`.
64 ```
65 root -b -q Fun4SimMicroDst.C
66 ```
67 
68 The last step is draw plots from Tree.
69 ```
70 root -b -q Fun4SimTree.C
71 ```
72 Various plots are outputted to `result/`.
73 
74 If all the commands ran fine, you delete `(5)` (or change to `(0)`) in the 1st step to use all DSTs.

76 ## Advanced Usage
77 
78 ### `cmake-this`
79 
80 You should execute this function when you add/delete files under `src/`.
81 It is a shell function defined in `setup.sh`.
82 It calls 'cmake' with a proper configuration.
83 It cleans up all files in the build & install directories.
84 
85 ### `make-this`
86 
87 You should execute this function when you modify files under `src/`.
88 It is a shell function defined in `setup.sh`.
89 It calls 'make install' with a proper configuration.
