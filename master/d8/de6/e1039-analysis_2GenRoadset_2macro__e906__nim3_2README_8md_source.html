1 # Macro to extract E906 NIM3 events
2 
3 This directory is to extract NIM3 events from the E906 digit files.
4 It outputs a small ROOT file named `bg_tree.root`, which will be used later in `macro_ana`.
5 
6 You usually need not carry out this step by yourself.
7 
8 
9 ## Setting of Event Extraction
10 
11 The setting of the event extraction is configured in `Fun4All.C` as usual.
12 The SubsysReco module that creates `bg_tree.root` is `src/SubsysRecoNim3.(h|cc)`.
13 You can modify them as you need.
14 
15 
16 ## Procedure
17 
18 The digit files had better be prestaged before use.
19 
20 ```
21 ../../HitEmbedding/macro_gen_emb_e906/prestage-input-file.sh
22 ```
23 
24 The generation process can/should be done on local computer, since one run takes only a few minutes.
25 It might be better to divide the whole process into multiple blocks like this;
26 
27 ``
28 ./gridsub.sh -n 1-100
29 ./gridsub.sh -n 101-1000
30 ./gridsub.sh -n 1001-2000
31 ./gridsub.sh -n 2001-4000
32 ``
33 
34 The run-by-run output files should be merged;
35 
36 ```
37 ./merge-output-file.sh
