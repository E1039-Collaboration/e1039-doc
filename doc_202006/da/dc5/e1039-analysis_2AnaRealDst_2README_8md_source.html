1 # e1039-analysis/AnaRealDst
2 
3 A compact program to analyze the "DST" file, which is created by the Main DAQ decoder.
4 Since it is yet under development and also depends on other software under development,
5 it might stop working occasionally. In such cases, please contact the author.
6 
7 ## Basic Usage
8 
9 The procedure below will read the DST file of run 202 (for example), which is
10 ```
11 /data2/e1039/dst/run_000202_spin.root
12 ```
13 and produce basic histograms and ntuple.
14 
15 1. source setup.sh
16 1. cmake-this
17 1. make-this
18 1. cd macro
19 1. root -b -q 'Fun4RealDst.C(202)'
20 1. display h1_ele_H1T.png &
21 
22 Here are some tips:
23 
24 - You need execute "source setup.sh" every time you open a new terminal.
25 - You need execute "make-this" every time you modify any file in "src/".
26 - You need execute "cmake-this" every time you add/delete any file in "src/".
27 - You can execute the macro at any directory.

29 ## Advanced Usage
30 
31 - The contents of the analysis are defined in class "AnaRealDst".
32  You can modify this class (i.e. "src/AnaRealDst.cc") to implement your own analysis.
33  - The E1039 data are structured by the SQ interface classes. To access them, you use several member functions of these classes. You can refer to the [oline document](https://e1039-collaboration.github.io/e1039-doc/annotated.html). Below are frequently-used classes;
34  - [SQRun](https://e1039-collaboration.github.io/e1039-doc/d7/db7/classSQRun.html)
35  - [SQEvent](https://e1039-collaboration.github.io/e1039-doc/d9/dd7/classSQEvent.html)
36  - [SQHitVector](https://e1039-collaboration.github.io/e1039-doc/d9/dbc/classSQHitVector.html)
37  - [SQHit](https://e1039-collaboration.github.io/e1039-doc/de/d79/classSQHit.html)
38  - [GeomSvc](https://e1039-collaboration.github.io/e1039-doc/d0/da0/classGeomSvc.html)
39 - The run ID and the number of events for analysis are defined in "macro/Fun4RealDst.C".
40  You can change them so that you need not give them in the command line.
