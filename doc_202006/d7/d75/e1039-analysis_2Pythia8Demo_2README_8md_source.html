16 ### Run simulation macro
17 ```
18 cd <path-to/seaquest-analysis/Pythia8Demo>
19 root -l -q Fun4Pythia8Demo.C //Pythia only or
20 root -l -q Fun4Sim.C //Geant4 simulation
21 ```
22 The output DSTRead.root file contanins a tree with TClonesArray branches.
23 TClonesArray may constains:
24 - [PHG4VtxPoint](https://haiwangyu.github.io/seaquest-offline-doc/d6/d81/classPHG4VtxPoint.html) for interaction vertex
25 - [PHG4Particle](https://haiwangyu.github.io/seaquest-offline-doc/de/dc9/classPHG4Particle.html) for truth particles
26 - [PHG4Hit](https://haiwangyu.github.io/seaquest-offline-doc/d3/d9e/classPHG4Hit.html) for Geant4 Hit
27 
28 You could use methods after loading the dinamic library.
29 The [ana.C](https://github.com/HaiwangYu/seaquest-analysis/blob/master/Pythia8Demo/ana.C) is a simple example looping over PHG4Particle TClonesArray.
30 Other TClonesArray branches can be accessed similarly.
